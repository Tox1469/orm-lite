# orm-lite

Mini ORM active record para SQLite.

## Instalacao

```bash
npm install orm-lite
```

## Uso

```ts
import * as lib from "orm-lite";
```

Veja `src/index.ts` para a API completa e exemplos de uso em producao.

## API

O pacote exporta classes e funcoes utilitarias com tipagem TypeScript completa. Consulte os arquivos `.d.ts` gerados em `dist/` apos `npm run build` para a referencia completa.
